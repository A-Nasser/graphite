Getting started

Graphite is powered by jQuery Mobile and comes with eight default colors.

  1. Download

    To grab Graphite, just download an archive or clone the github repo. Graphite comes with a set of pre-made themes in the "generated" folder based around three simples colors.

  2. Install

    To add a Graphite theme to your jQuery Mobile 1.2.0 project, just copy the theme CSS over in place of the jQuery Mobile default CSS file.

  3. Customize

    You can use our pre-made themes or even build your own by using the helper/painter.py script and the settings in themes/yourtheme.yaml.
